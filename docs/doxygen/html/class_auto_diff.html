Detailed Description

A node in the AutoDiff computation graph.

Each instance of AutoDiff represents a node in the AutoDiff computation graph. Complicated functions can be composed by chaining multiple nodes. Each node has a function value – the value of the function. Each node also has a derivative value – the derivative of the function at the current value.
