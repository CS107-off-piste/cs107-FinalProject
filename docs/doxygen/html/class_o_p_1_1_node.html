void(* OP::Node::_backward_func_ptr) (Node &this_node)

A function pointer that defines the behaviour of this Node in reverse mode. For example, if backward_func_ptr is set to OP::mul_backward() then this Node is a multiplication node that adds its child nodes when computing this Node's value and derivative in forward mode.
